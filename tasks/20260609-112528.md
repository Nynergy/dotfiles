# [isli] Task index printing with padded digits

- Created:  June 09, 2026
- Status:   OPEN

When isli prints tasks, it prints the index before the task file. This index
should be padded keeping the total number of tasks in mind.

For example, if there are 25 tasks to list, that means we need 2 digits for the
task index. A task with index 6 should be padded to ' 6' when printed, similar
to how we accomplish a similar padding in rimeglass.
